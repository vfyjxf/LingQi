### Step A: 逐文件 diff 提取函数

   - 使用正则 `/(?=^diff --git )/m` 将 diffText 按 `diff --git` 边界拆分为多个文件块
   - 遍历每个块，用 `/^diff --git a\/(.+?) b\/(.+?)$/m` 提取文件路径
   - 若路径与 `filename` 匹配（比较 `a/` 后的路径或 `b/` 后的路径），返回该完整块
   - 若无匹配，返回 `""`

     // 文件筛选（新增）：selectedFile 非空时只保留该文件的隐患

     // 级别/类别筛选（保留现有逻辑）
     if (activeFilter.type === "severity") return r.severity === activeFilter.value;
     if (activeFilter.type === "category") return r.category === activeFilter.value;
     return true;
   });
